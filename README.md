# 我的图形学渲染器 🚀

> 从零构建的实时/离线渲染引擎，不依赖 OpenGL/DirectX

## ✨ 核心特性

- **光栅化管线**：MVP变换、Z-Buffer、SSAA抗锯齿
- **高级着色**：Phong光照、法线贴图、双线性滤波
- **光线追踪**：BVH加速结构、PBR材质、全局光照

## 🛠️ 技术栈

- C++17 / Eigen3 / OpenCV
- 算法：Möller–Trumbore求交、重心坐标插值
