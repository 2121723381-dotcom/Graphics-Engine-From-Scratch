# 项目对比：光线追踪 ， 软光栅化

## 核心逻辑差异（非架构整合，纯技术对比）

| **维度**         | 项目一：光线追踪 (project1_raytracer)       | 项目二：软光栅化 (project2_rasterizer)   |
|------------------|------------------------------------------|----------------------------------------|
| **核心目标**     | 实现物理真实的全局光照                   | 验证 GPU 渲染管线底层逻辑              |
| **关键算法**     | BVH 加速 + 漫反射/镜面反射递归           | 三角形遍历 + Z-Buffer + 重心坐标插值   |
| **我的理解**     | `光线求交是性能瓶颈，BVH 构建质量直接影响10倍以上速度差异` | `光栅化本质是2D投影问题，Z-Buffer解决深度冲突是核心` |
| **课程作业关联** | Games101 作业5-6（光线追踪基础）         | Games101 作业2-3（MVP变换与光栅化）    |
